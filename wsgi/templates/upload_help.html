Download Claim Template File

Your file must be in .CSV format or your claim will be rejected. Excel has an option to "Save As" a .csv file but will give a warning about loss of Excel functionality before doing so. See the sample warning message below

Just ignore this warning and answer "Yes" to save in .CSV format.

Please note also that your Excel spreadsheet should conform to the layout of the sample spreadsheet provided. That means the column names must all match and must all be in the same order. The column names should be on the first line of the spreadsheet and your data should start on the second line with no gap.
